# git上传配置
## 1 git init  
    初始化git配置文件
## 2 git add .   
    提交所有的文件更新
## 3 git remote add origin 地址   
    添加远程提交的地址  修改origin名称可以添加多个仓库地址
## 4 git branch  -M main  
    提交分支  选择mian分支  还可以是master分支
## 5 git commit -m "内容"
    提交更新信息
## 6 git push  origin 
    提交到远程仓库

### 可以直接修改git文件夹下的config文件的分支文件
    添加一个url = https://github.com/momeng9/document.git
    然后直接提交就可以了
