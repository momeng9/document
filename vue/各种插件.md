## 1 拖拽插件
    安装vue.draggable  这是VUE2.7以前的版本适用
    npm i -S vuedraggable
    2.7及其以上版本我安装 ，vue-draggable-plus  支持 Vue>=v3 或 Vue >=2.7 
    npm install vue-draggable-plus
